## ホーム画面
ホーム画面では、以下の機能にアクセスできます。
1. **ユーザー名の表示**: 登録されたユーザー名が表示されます。
2. **今日の目標**: 現在の挑戦回数や目標達成状況を確認できます。
3. **カレンダー**: 今日の目標を達成するとチェックマークで表示されます。

---  

## カレンダー機能
ホーム画面にあるカレンダーでは、目標を達成した日が **緑のチェックマーク** で表示されます。また、1日の目標回数を達成した場合には **赤のチェックマーク** が表示されます。

### 機能一覧
- 達成済みの日を確認
- 今日の日付を強調表示
- 日付をタップして詳細を表示

---  

## 今日の目標
「今日の目標」では以下の情報が表示されます：
- 現在の挑戦回数
- 毎日の目標

目標を達成すると、**緑色のチェックマーク** が表示されます。

## メニュー画面
画面の下部にメニューから、以下の機能にアクセスできます：
1. **問題**: レベルを選択して問題を解く画面に遷移します。
2. **成績**: 進捗状況や成績をグラフで確認できます。
3. **復習**: 過去に学習した問題の復習ができます。
4. **挑戦**: 挑戦モードでは、ゲーム感覚で問題に挑戦できます。

### メニューアイコンの色
- 問題: 白色
- 成績: ピンク
- 復習: 黄色
- 挑戦: 青色

## レベル選択画面

問題を解く前に「レベル選択画面」表示します。この画面では、１年生から６年生までの学年レベルを選択でき、選択したレベルに基づいて適切な問題が表示されます。

## 各学年の問題詳細

### １年生
- **たしざん（1ケタ、2ケタ）**: 1ケタと2ケタの数字を使った基本的な足し算の練習。
- **たしざんのあなうめ**: 足し算の一部が空欄になっており、正しい答えを埋める問題。
- **おはなしもんだい（たしざん）**: 足し算を含む簡単な文章問題。
- **ひきざん（1ケタ、2ケタ）**: 1ケタと2ケタの数字を使った基本的な引き算の練習。
- **ひきざんのあなうめ**: 引き算の一部が空欄になっており、正しい答えを埋める問題。
- **おはなしもんだい（ひきざん）**: 引き算を含む簡単な文章問題。

---

- **足し算（3ケタ）**: 3ケタの数字を使った足し算の練習。
- **引き算（3ケタ）**: 3ケタの数字を使った引き算の練習。
- **2ケタの文章問題（足し算）**: 足し算を含む文章問題。
- **2ケタの文章問題（引き算）**: 引き算を含む文章問題。

- **掛け算の文章問題**: 掛け算を含む文章問題。
- **割り算の文章問題**: 割り算を含む文章問題。

## 復習画面
問題モードをプレイ後、成績画面に出てくる各問題の横のチェックマークをタップするとその問題が保存されるようになっています。

---

## 挑戦モード
「挑戦モード」では、ゲーム感覚で問題に挑戦できます。ユーザーは以下の3つのチャレンジから選ぶことができます：

1. **タイムアタック** 時間内にどれだけ多くの問題を解けるか挑戦します。素早く正確に解答し、タイムを縮めましょう。

2. **ナンバープレート** ４つの数字がランダムに生成され、入力元にある四則演算と数字を利用して答えを「１０」になるように数式を作りましょう。

3. **四則演算対戦** 足し算、引き算、掛け算、割り算などの問題を他のプレイヤーと対戦しながら解くモードです。対戦相手を倒し、ランキングを上げましょう。

### 各チャレンジの詳細
- **タイムアタック**: 制限時間内にできるだけ多くの問題を解答するタイムチャレンジ。
- **ナンバープレート**: 数字と四則演算記号を使って数式を作っていく。
- **四則演算対戦**: 対戦モードで他のプレイヤーと四則演算を競い合う。

## Flutter の環境設定

### Flutter SDK のインストールと確認方法
私たちはAndroidStudioでFlutterを使用したので、その方法を下記に記述します。

１．Flutter公式サイト（https://flutter.dev）

２．「Get Started」→「Install」からLinuxを選択

３．SDKのダウンロードしたら解凍。

４．ターミナルを開き、下記のコマンドを実行して環境変数にFlutterのパスを追加します。
```
echo 'export PATH="$PATH:/path/to/flutter/bin"' >> ~/.bashrc 
source ~/.bashrc
```
５．Flutterが動作するか角煮
```
flutter doctor
```
### Dart & Flutter

１．AndroidStudioを開いてPluginsを開く

２．MarketPlaceでFlutterを検索して、インストール

３．AndroidStudioを再起動

### Flutterプロジェクト作成

１．AndroidStudioを開き、New Flutter Project を選択。

２．プロジェクトの種類を「Flutter Application」に設定

３．Flutter SDKのパスを指定

４．プロジェクト名と保存先ディレクトリを設定してFinishをクリック。

### 動作確認
```
flutter run
```
